# Create the JSON file

Use the following as a template for your json file

```
[
{
  "name": "480p0",
  "fps": 0,
  "bitrate": 1600000,
  "width": 854,
  "height": 480,
  "profile": "h264constrainedhigh",
  "gop": "1"
},
{
  "name": "720p0",
  "fps": 0,
  "bitrate": 3000000,
  "width": 1280,
  "height": 720,
  "profile": "h264constrainedhigh",
  "gop": "1"
},
{
  "name": "1080p0",
  "fps": 0,
  "bitrate": 6500000,
  "width": 1920,
  "height": 1080,
  "profile": "h264constrainedhigh",
  "gop": "1"
}
]
```
